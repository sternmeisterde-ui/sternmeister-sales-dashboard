## Какие поля берутся из Kommo

| Поле | Откуда | Для чего |
|---|---|---|
| Контакт (имя, ID) | `/contacts?query=телефон` | Привязка к контакту |
| Сделка (ID, название) | Через контакт → leads | Привязка к сделке |
| Воронка | `lead.pipeline_id` | Определение отдела (R2/D2) |
| Стадия | `lead.status_id` | Определение типа оценки |
| Ответственный | `lead.responsible_user_id` | Привязка к менеджеру |
| Ссылка на CRM | `leads/detail/{leadId}` | Ссылка в дашборде и Telegram |
| Custom fields | Только для D2 | Доп. информация по клиенту |

### Поиск контакта по телефону

1. Ищем контакт по номеру → берём все его сделки
2. Если сделок нет → ищем сделки напрямую п�� номеру
3. Из всех сделок выбираем лучшую:
   - Активная в известной воронке (приоритет)
   - Любая в известной воронке
   - Любая активная
   - Последняя по дате
