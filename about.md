# О Проекте (About)

## Описание
Дашборд для руководителей отделов продаж, интегрированный с Kommo CRM и серверлесс-базой данных Neon PostgreSQL.
Система предназначена для мониторинга показателей двух отделов продаж: "Госники" (B2G) и "Коммерсы" (B2B).

## Основные Экраны и Функционал
1. **Общий дашборд продаж:**
   - Данные из Kommo CRM: общая сумма продаж, лучшие показатели (время звонков, кол-во звонков, время на линии).
2. **Аналитика "Реальные звонки":**
   - Переключение между отделами ("Госники" / "Коммерсы").
   - Сводные данные по звонкам.
   - Детализация по каждому менеджеру.
   - Таблица звонков (аватарка, имя, время, транскрибация, ссылка на Kommo, скоринг качества, аудио-плеер).
3. **Аналитика "Ролевые звонки с AI агентом":**
   - Та же структура, что и для реальных звонков, но с фокусом на тренировочные (ролевые) звонки с искусственным интеллектом.

## Технологический Стек
- **Frontend / Backend:** Next.js (App Router), TypeScript
- **База Данных:** Neon (Serverless PostgreSQL)
- **ORM:** Drizzle ORM
- **Стилизация & UI:** Tailwind CSS, Shadcn UI
- **Графики:** Recharts

## Интеграции
- Kommo CRM (API/Webhooks для получения сделок, контактов, данных по звонкам)
- AI Сервис (Для транскрибации и скоринга аудио)
